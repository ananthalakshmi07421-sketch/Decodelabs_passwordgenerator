# 🔐 Password Generator (Python)

A simple command-line Password Generator built with Python. This application generates strong, random passwords based on the user-specified length while ensuring the password contains lowercase letters, uppercase letters, numbers, and special characters.

## 📌 Features

- 🔑 Generates secure random passwords
- 🔠 Includes uppercase and lowercase letters
- 🔢 Includes digits
- ✨ Includes special characters
- 📏 User-defined password length
- ⚠ Validates minimum password length (4 characters)
- 🖥 Easy-to-use command-line interface

## 🚀 How to Run

### 1. Clone the repository
